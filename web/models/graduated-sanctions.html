1 Core Design Principles for the Efficacy of Groups ↑

Model element: Graduated Sanctions

When group norms are not followed, then sanctions should be applied, but should be done in a graduated fashion. For example, sanctions might start as friendly pressure from peers, but then may proceed to more serious consequences if negative behavior persists.
